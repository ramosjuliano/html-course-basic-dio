Head

No "Head" traremos para aplicação aquilo que será carregado antes e preparará o conteúdo para o usuário.

No nosso caso é bem simples a trás pequenas configurações e o título da página, que estará exibido no título da aba.

Body

O "Body" já é responsável por trazer as informações que serão exibidas ao usuário.

Essas informações e sua organização serão tratadas no decorrer da página.

Títulos "h1" -> "h6"

Os títulos são tags fundamentais para organização do seu arquivo HTML. Nelas organizaremos nossos conteúdos de acordo com a sua hierarquia dentro do nosso documento.

Nossas tags são ordenadas de acordo com o seu número. Sendo 1 o título de mais relevância e 6 o de menor relevância.

Para facilitar, siga a seguinte lista ordenada de hierarquia (Do maior ao menor).

  1. h1 - Maior relevância
  2. h2
  3. h3
  4. h4
  5. h5
  6. h6 - Menor relevância

Parágrafos

A tag "p" é utilizada para delimitar parágrafos dos conteúdos do nosso documento HTML. Ele tem papel fundamental na organização de marcação de contexto.

Formatações Básicas

Existem diversos tipos de formatações que podem ser utilizadas diretamente no HTML sem necessidade de delegar para o CSS. Algumas delas citaremos aqui!

Negrito

O texto destacado entre as tags "strong" são formatadas como negrito

Itálico

O texto destacado entre as tags "i" são formatados como itálico

Sublinhado

O texto destacado entre as tags "u" são formatados como sublinhado

Texto Pequeno

O texto destacado entre as tags "small" são formatadas num texto pequeno

Marcação

O texto destacado entre as tags "mark" são formatadas como uma espécie de marca texto

Listas Ordenadas e Listas Desordenadas

A listas no HTML são divididos em dois subgrupos. As listas ordenadas, que devem ser usadas para listagem de itens com graus determinados de relevância e ordem. As listas Desordenadas já devem ser usadas em situações de divisões sem critérios definidos.

Abaixo apresentaremos os dois exemplos

Listas Ordenadas

A lista de classificação de um campeonato é ordenada do melhor para o pior.

  1. Real Madrid
  2. Barcelona
  3. Porto
  4. Paris Saint Germain
  5. Milan
  6. Ajax

Listas Desordenadas

Uma lista de compras não exige uma ordem...

  • Ovos
  • Leite
  • Arroz
  • Feijão
  • Massa

Links e Navegabilidade

Os links ou tags "a" podem levar o usuário da página para referências dentro da própria página, outras páginas do mesmo recurso ou, até então, outras páginas externas.

Âncora

A Âncora vai nos levar para outra parte da mesma página com base na referência estabelecida nessa conexao. No nosso caso, voltaremos para o índice clicando aqui.

Outra página - Interno

O link interno é já é um recurso utilizado para circular entre páginas do mesmo produto. Nesse caso o recurso é o mesmo "a", porém usamos "_self" para o atributo target. Nesse caso a aplicação usará a mesma aba para acessar os recursos, exatamente como ao clicar aqui aqui.

Outra página - Externo

Já o link externo leva o usuário para uma página fora da aplicação. Nesse exemplo vou levá-los ao site do Google. Nesse caso usaremos "_blank" no atributo target para que a aplicação abra uma nova aba e mantenha nosso portal aberto na página atual.

Outras Tags

Nessa etapa farei uma passagem rápida por outras tags também utilizadas no HTML.

Tag "hr"

Essa tag gera uma linha que serve como separação na página.

Tag "sub"

O sub é aquele textinho que fica pequeno após determinado termo. O exemplo que lembro é o logaritmo... Log 5 2.

Tag "sup"

Já a tag "sup" é aquele clássico relacionado a referência no texto que será exibido no rodapé da página. Olha para essa referência 2."

Tag "del"

É utilizado no HTML para "rasurar" um texto determinado. No geral é utilizado para deixar explícito que determinado texto não é mais válido e manter seu histórico explícito. Muito utilizado em atualização de textos de leis onde determinados pontos foram modificados ou excluídos

A renovação da carteira de motorista é de 5 anos.

Tag "abbr"

Seu uso é simples. Ele é usado em palavras abreviadas, onde é possível incluir seu significado no atributo e visualiza-lo ao posicionar o mouse sob a palavra em questão...

Um exemplo bem interessante é o significado de HTML.
